Resource Class

Defined in: sdk/model/model.factory.js:166
Module: model
Parent Module: AP

A Resource is nothing more than an object with a $save method. This class is defined dynamically so that whenever a Model creates a Resource, the resources's $save method will have access to the "save" method of the specific Model that created it
